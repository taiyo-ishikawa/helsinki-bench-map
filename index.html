Data Sources

Helsinki YLRE (City Registry)
Retrieved via HRI WFS API (YLRE_Katuosat_piste, YLRE_Viherosat_piste). License: CC BY 4.0

OpenStreetMap
Fetched via Overpass API (amenity=bench). License: ODbL 1.0

Population & income
Statistics Finland Paavo 2023 (postal code stats, pno_tilasto_2023) aggregated to kaupunginosa by centroid spatial join.

Classification

City-registered — YLRE official registry (street & park). Includes YLRE+OSM matched pairs (5 m threshold).

Community-mapped — OSM benches absent from YLRE registry.

Nature / Park — YLRE park layer benches or OSM benches within park/forest polygons.

Waterfront — Within 200 m of the Baltic Sea coastline (OSM coastline data).

Methodology

Data collected April 2026. Finnish → English translation. EPSG:3879 → WGS84 reprojection (pyproj).

Deduplication: 5 m threshold (scipy cKDTree). 3,040 merged YLRE+OSM pairs.

District stats: Areas in EPSG:3067. Population aggregated from 84 Helsinki postal code areas (44/60 districts matched).
