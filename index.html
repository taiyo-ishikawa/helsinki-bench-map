Data Sources

Helsinki YLRE (City Registry)
Register of Public Areas of the City of Helsinki, maintained by the Urban Environment division. Retrieved via Helsinki Region Infoshare (HRI) using the WFS API (YLRE_Katuosat_piste for street benches, YLRE_Viherosat_piste for park benches). License: CC BY 4.0

OpenStreetMap (OSM)
Volunteer-contributed geographic data retrieved via Overpass API using the tag amenity=bench within the Helsinki bounding box (60.10–60.35°N, 24.82–25.25°E). License: ODbL 1.0

Classification

City-registered — Benches recorded in the Helsinki YLRE official public areas registry, covering street areas and green/park areas. Also includes benches confirmed by both YLRE and OSM (matched within 5 metres); the YLRE record is treated as authoritative.

Community-mapped — Benches present in OpenStreetMap but absent from the YLRE registry. These may be located at private properties, transit facilities, or outdoor areas not yet recorded by the city. OSM data may include additional attributes such as backrest presence, material, and seat count.

District Boundaries

District (kaupunginosa) assignment was performed by spatially joining each bench point to the Kaupunginosajako polygon layer from the Helsinki WFS service.

Methodology

Data was collected in April 2026. Finnish field values were translated to English. Coordinates were reprojected from the Helsinki local CRS (EPSG:3879) to WGS84 (EPSG:4326) using pyproj.

Deduplication: Benches appearing in both YLRE and OSM were identified using a 5-metre proximity threshold (scipy cKDTree) and merged into a single feature retaining attributes from both sources.

Total: 12,752 unique bench locations across 56 districts after deduplication (original: 8,852 YLRE + 6,776 OSM = 15,628 records; 3,040 matched pairs merged).

Source code and data: github.com/taiyo-ishikawa/helsinki-bench-map
