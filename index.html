Data Sources

Helsinki YLRE (City Registry)
Register of Public Areas of the City of Helsinki, maintained by the Urban Environment division. Retrieved via Helsinki Region Infoshare (HRI) using the WFS API (YLRE_Katuosat_piste for street benches, YLRE_Viherosat_piste for park benches). License: CC BY 4.0

OpenStreetMap (OSM)
Volunteer-contributed geographic data retrieved via Overpass API using the tag amenity=bench within the Helsinki bounding box. License: ODbL 1.0

Classification

City-registered — Benches in the Helsinki YLRE official registry (street & park areas). Includes benches confirmed by both YLRE and OSM (matched within 5 m).

Community-mapped — Benches in OpenStreetMap not found in the YLRE registry. May include private, transit, or unregistered locations.

Nature / Park — YLRE park benches (YLRE_Viherosat_piste layer) or OSM-only benches within park/forest polygons.

Waterfront — Benches within 200 m of the Baltic Sea coastline (OSM coastline data).

District Boundaries

District assignment via Point-in-Polygon with the Kaupunginosajako WFS layer. District areas calculated in EPSG:3067.

Methodology

Data collected April 2026. Finnish field values translated to English. Coordinates reprojected from EPSG:3879 → WGS84 using pyproj.

Deduplication: Benches in both YLRE and OSM merged using a 5 m threshold (scipy cKDTree). Merged records retain attributes from both sources.

Total: 12,752 unique bench locations across 56 districts (original: 8,852 YLRE + 6,776 OSM = 15,628 records; 3,040 matched pairs merged).

Source code & data: github.com/taiyo-ishikawa/helsinki-bench-map
