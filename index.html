Data Sources

Helsinki YLRE (City Registry)
Retrieved via HRI WFS API (YLRE_Katuosat_piste, YLRE_Viherosat_piste). License: CC BY 4.0

OpenStreetMap
Fetched via Overpass API (amenity=bench). License: ODbL 1.0

Population & income
Statistics Finland Paavo 2023 (postal code area statistics, pno_tilasto_2023), aggregated to kaupunginosa by centroid spatial join. License: CC BY 4.0

Classification

City-registered — YLRE official registry (street & park). Includes YLRE+OSM matched pairs (5 m threshold).

Community-mapped — OSM benches absent from YLRE registry.

Nature / Park — YLRE park layer (YLRE_Viherosat_piste) or OSM benches within park/forest polygons.

Waterfront — Within 200 m of the Baltic Sea coastline (OSM coastline data, scipy cKDTree).

Income indicator

hr_mtu: median personal income of district residents (€/year). District value = population-weighted average of constituent postal code medians.

Methodology

Data collected April 2026. Finnish → English field translation. EPSG:3879 → WGS84 reprojection (pyproj).

Deduplication: YLRE + OSM matched within 5 m (scipy cKDTree). 3,040 merged pairs.

District areas: Calculated in EPSG:3067 (Finnish national metric CRS) from Kaupunginosajako polygon layer.

Total: 12,752 bench locations across 56 districts (from 15,628 raw records).

Source code & data: github.com/taiyo-ishikawa/helsinki-bench-map
